# 貪食蛇遊戲

這是一個使用 Python 和 Pygame 製作的經典貪食蛇遊戲。

## 安裝步驟

1. 確保您已安裝 Python 3.6 或更高版本
2. 安裝必要的依賴：
   ```
   pip install -r requirements.txt
   ```

## 運行遊戲

在終端機中執行以下命令：
```
python snake_game.py
```

## 遊戲控制

- 使用方向鍵（↑↓←→）控制蛇的移動方向
- 吃到紅色食物可以增加分數和蛇的長度
- 撞到自己會重新開始遊戲
- 按視窗的關閉按鈕可以退出遊戲

## 遊戲特點

- 穿牆功能：蛇可以從一邊穿過到另一邊
- 即時分數顯示
- 流暢的遊戲體驗
- 簡潔的視覺設計
